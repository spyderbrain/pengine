How to download and install

First, ask yourself whether you really need to download and install the Pengine platform at this time. The platform is designed to allow online interaction with Prolog running in the cloud, and if you just want to check it out, you can probably do so without downloading and installing your own instance of the Pengine server. Indeed, you can skip this entire page and quickly move on to the fun parts, the live Tutorial, the Scratchpad and the SWISH application. The only area that you will not have access to is the password protected Pengine Administrator.

If on the other hand you want to develop a serious application on top of the Pengine platform, you will need access to the apps directory and the Administrator pages, and thus need to download and install the whole distribution. Allowing access to the Pengine server from localhost only is of course safe, but if you consider opening up to the rest of the world, please consider the following safety information:

We cannot at this time guarantee the safety of the Pengine platform. We think it is safe, but only if you know what you are doing. You run it at your own risk!

Install Pengine

Basic installation merely requires installing a recent version of SWI-Prolog and unpacking the Pengine platform below.

Start run.pl as ./run.pl on Unix or double-click run.pl on Windows and direct your browser to the indicated address (default is http://localhost:5003/). You will be asked to enter a user name and a password, and if this is the first time you use Pengine you should use "admin" for both. You will end up on the Apps settings page.

You most likely want to change the admin password right away. Clicking the Account link will get you to a page that will allow you to do just that.

The Pengine Administrator

The Pengine Administrator is a password protected area that allows the owner of a Pengine server instance to change the settings of applications as well as of the server itself. It also features a statistics page where statistics of the workings of the systems is presented as well as an account page where the admin password can be changed.

App settings

The Pengine Administrator's App settings page is where applications can be individually activated and deactivated at will. It furthermore has an important role concerning the safety of the system as a whole since this is where you get an indication of safety issues related to the predicates introduced by new applications, predicates exported by the index.pl file in the apps directory.
